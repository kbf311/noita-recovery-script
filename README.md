# Noita セーブデータ復元スクリプト

このスクリプトは、ローグライクゲーム『Noita』で死亡時にセーブデータが消去される仕様に対し、「ある程度の進行データを引き継いだ状態で、最初からゲームを再スタートする」ためのバックアップ・復元ツールです。

> [!NOTE]
> **※本スクリプトは、制作者が自分自身のプレイ環境や好みのゲームバランス（強くなりすぎない調整など）に合わせて開発した、自分専用のツールです。**
> 動作保証や汎用的な設定カスタマイズ等は考慮されていません。

強くなりすぎないように、**HPや初期位置はゲーム開始時の状態（初期値）にリセット**しつつ、所持品や一部の特典（パーク）などを引き継いでニューゲームを開始できます。

---

## 💡 できること（仕様）

*   **強さを抑えたニューゲーム**
    *   バックアップ時点の**所持杖、呪文、インベントリ、プレイヤーのパークやステータス**を引き継ぎます。
    *   ただし、ゲームバランス崩壊を防ぐため、最大HPは初期値（100）にリセットされます。
    *   スタート位置は最初の洞窟の入り口（初期座標）に戻ります。
*   **安全設計と世代管理**
    *   バックアップは自動的に**最大5世代分**保存されるため、誤って上書きしても過去のデータから復元可能です。

---

## ⚠️ 重要：実行時の注意点

> [!IMPORTANT]
> スクリプトを実行する際は、**必ずNoitaのゲーム本体を完全に終了（ゲームを落とした状態）させてから**実行してください。
> ゲームが起動したまま実行すると、警告メッセージが表示されて処理が中断されます。

---

## 🛠 使い方

### 1. バックアップを取る（データをセーブしたい時）
「この状態を保存して、死んだらここから引き継いでやり直したい」というタイミングで実行します。

1.  **Noitaを終了します（ゲームを完全に落とします）。**
2.  `Run-Backup.bat` をダブルクリックして起動します。
3.  画面に「バックアップが完了しました」と表示されたらキーを押して閉じます。
    *   スクリプトと同じ場所に `xml` フォルダが作成され、バックアップデータが保存されます。

### 2. 復元する（死んでしまい、データを引き継いでやり直したい時）
死んでしまってセーブデータが消えた後、引き継ぎデータで最初から始める際に実行します。

1.  **Noitaを起動し、「New Game」で新しいゲームを開始します。**
    *   ※セーブデータフォルダ（`save00`）を新しく生成させるために一度開始する必要があります。
2.  ゲームが始まったら、**一度Noitaを終了（ゲームを落とした状態に）します。**
3.  `Run-Restore.bat` をダブルクリックして起動します。
4.  画面に「復元が完了しました」と表示されたらキーを押して閉じます。
5.  Noitaを起動し、「**Continue**」を選択して再開します。
    *   初期位置（洞窟前）＆初期HP（100）の状態で、バックアップ時のアイテムや一部パークを引き継いでスタートします。

## 📁 ファイル構成

*   [Run-Backup.bat](./Run-Backup.bat)：バックアップを起動するバッチファイル
*   [Run-Restore.bat](./Run-Restore.bat)：復元を起動するバッチファイル
*   [Backup-Noita.ps1](./Backup-Noita.ps1)：バックアップ処理の本体（PowerShellスクリプト）
*   [Restore-Noita.ps1](./Restore-Noita.ps1)：復元・データ調整処理の本体（PowerShellスクリプト）
*   `xml/`：バックアップされたデータが保存されるフォルダ（自動生成）
